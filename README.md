# Personal Expense Tracker (PET) System

## Overview
The **Personal Expense Tracker (PET)** is a web-based application developed using **HTML, CSS, and JavaScript** that allows users to efficiently manage their daily expenses. The system is designed to help users track, categorize, and analyze their spending habits over time. It is user-friendly, visually appealing, and modular in design.

---

## Features
- **Add, Edit, and Delete Expenses**: Users can record new expenses, update existing ones, or remove unwanted entries.
- **Categorize Expenses**: Track spending across different categories such as Food, Transport, Utilities, Entertainment, and more.
- **Expense Summaries**: Generate weekly or monthly summaries of expenses for better financial insights.
- **Visual Representation**: Data is presented using charts or tables to give a clear view of spending patterns.
- **Responsive Design**: Works well on desktop and mobile devices with a clean, professional blue and white theme.

---

## Technologies Used
- **HTML5**: For structuring the web pages.
- **CSS3**: For styling and creating a modern responsive design.
- **JavaScript**: For handling application logic, managing expenses, and rendering charts.
